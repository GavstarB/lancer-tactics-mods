## Old Versions
Older mod versions, found in the `/old/` directory.

- **gms-1st-party-1.0.pck** *(v0.4.7)* - no icons, crash when using Balance Control Lockout on a grunt.
- **gms-1st-party-1.1.pck** *(v0.4.7 - v0.5.2)* - no text highlighting.
- **gms-1st-party-1.2.pck** *(v0.4.7 - v0.5.2)* - broken on 0.6.0.

- **ipsn-caliban-1.0.pck** *(v0.4.7)* - version compatible with 0.4.7.
- **ipsn-caliban-1.1.pck** *(v0.5.0 - v0.5.2)* - no text highlighting, Flayer Shotgun is accurate instead of inaccurate.
- **ipsn-caliban-1.2.pck** *(v0.5.0 - v0.5.2)* - Spike Grenade knockback is directional, Flayer Shotgun is inaccurate instead of accurate, walking armoury is bugged when adjacent.
- **ipsn-caliban-1.3.pck** *(v0.4.7 - v0.5.2)* - broken on 0.6.0, Rapid Maneuver Jets have no heat cost. 

- **ipsn-zheng-1.0.pck** *(v0.4.7 - v0.5.2)* - broken on 0.6.0, Molten Wreath can hit the same target multiple times.

- **ssc-alts-1st-party-1.0.pck** *(v0.4.7 - v0.5.2)* - no text highlighting, orchis can use Royal Guard when slowed.
- **ssc-alts-1st-party-1.1.pck** *(v0.4.7 - v0.5.2)* - broken on 0.6.0.

- **mhi-1.0.pck** *(v0.4.7)* - no icons.
- **mhi-1.1.pck** *(v0.4.7 - v0.5.2)* - no text highlighting.
- **mhi-1.2.pck** *(v0.4.7 - v0.5.2)* - broken on 0.6.0.
